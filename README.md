# Tashkent Real Estate Market Analysis 🏠📊

This project provides a comprehensive data analysis of the real estate market in Tashkent, Uzbekistan. The analysis is performed using Python and common data science libraries.

## 🚀 Project Overview
The goal of this project is to understand the factors affecting house prices in Tashkent, identify the most expensive districts, and visualize the correlation between house features.

## 🛠️ Tools Used
* **Python** (Pandas, NumPy)
* **Visualization:** Matplotlib, Seaborn
* **Environment:** PyCharm Professional / Jupyter Notebook

## 📈 Key Insights
* **Top District:** Mirobod district has the highest average property prices.
* **Most Active:** Chilonzor district has the highest number of listings.
* **Price Driver:** Property size (`size`) is the strongest factor influencing price (Correlation: 0.78).
* **Affordability:** Bektemir district remains the most affordable area.

## 📂 Project Structure
* `data/`: Contains the raw `uybor.xlsx` dataset.
* `notebooks/`: Contains the `tashkent_analysis.ipynb` with full code and visualizations.
* `README.md`: Project documentation.
